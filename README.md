# dbimp

## Objectif

Import (and export) data engine in a database

## Building

On linux console (like wsl2 too), install java tools

~~~bash
sudo apt install -y openjdk-17-jdk maven
clone https://github.com/mpatron/dbimp.git
cd dbimp
mvn package
~~~
